======================
Offline install script
======================

install packges offline from repository file/install package.

Setup
=====
	1. Extract the file "mini_repo_v1.tar.gz".

Usage
=====
	1. Open terminal
	2. Run:
		bash install.bash
		 and then enter password	
	3. Run:
		bash soi-install.bash
		 and then enter password
